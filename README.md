# ImagEHR
GenAI Genesis 2025

# Install PyTorch
https://pytorch.org/get-started/locally/

# Setup
```
pip install -r requirements.txt
```

# Pull submodules
```
git submodule update --init --remote --recursive
```

# Environment
Copy `.env.example` into `.env` and fill in your API keys, etc.

# Run the server
```
python main.py
```
